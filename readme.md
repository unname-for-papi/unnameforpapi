# unname for papi

**this addons to papi is for people who making worlds like servers**


**adding features**
```
%unnamepapi____worldonlineadding____<world1>____<world2, you can like inf of adding>%
```

**for get many players are there in the world**

```
%unnamepapi____itemcount____<itemname>%
```

**Shows how many Specific item the player has**

```
%unnamepapi____diedsworld____<world1>____<world2, you can like inf of adding>%
```

**Shows How many times has been dieded in Specific worlds**

```
%unnamepapi____killsworld____<world1>____<world2, you can like inf of adding>%
```

**Shows How many times has been kills in Specific worlds**

```
%unnamepapi____spenttimeworldtemplate____<Default or custom>____<world1>____<world2, you can like inf of adding>%
```

**Player Time Spent in Specific World**

```
%unnamepapi____spenttimeworld_<S for seconds,M for minutes,H for hours,D for days>____<world1>____<world2, you can like inf of adding>%
```

**Player Time S or M or H or D Spent in Specific World**

```
%unnamepapi____serveronline%
```

**Show how many players are on the server**

Why 4 underscores?
Because world names can contain _ (like world_nether), this addon uses four underscores (____) as a separator to avoid conflicts.
